# Backbone boilerplate
Backbone boilerplate will help you up and run a complete Frontend dev environment

## The Tools
- Backbone
- Coffeescript
- Less
- Grunt
- Browserify
- Bower
- Node Express server

## Installation
- clone
- `cd backbone-presentation`
- `npm install`
- `grunt`

## Running
- `npm start`
- open browser in localhost:8080
